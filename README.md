# 内容
授業で制作したコードになります。  
OpenCVを用いて、提供された動画に写る魚の動体検知を行うプログラムです。 
まず、最初のフレームを画像として取り込みます。
その後、各魚の色ごとに設定した閾値に応じて魚を白、それ以外を黒に2値化します。
次に、各フレームごとに最初のフレームとの差を計算します。
その差から、動体を検出し、動体を白、それ以外を黒に2値化します。
最後に、作成した2値化画像同士で論理積をとることで、動いていてかつ特定の色の物体を検出することで魚を検出するといったコードになります。
